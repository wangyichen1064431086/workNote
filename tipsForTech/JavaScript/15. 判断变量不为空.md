执行：

```
  if(x) {
    console.log('true');
  } esle {
    console.log('false');
  }
```
### x值各情况对应运行结果

x   | 结果
----|-----
' ' |'false'
0  |'false'
false|'false'
[] |'true'
{} |'true'
new String()|'true'
new Object()|'true'
new Array()|'true'

### 总结
当x为对象时，即是对象为空，也满足x==true。

另外 数组、对象都是对象，以new String()形式创建的字符串也是对象。
