## 三. 验证常用正则表达式

### 1. 邮箱

简化版：
```s
 /\S+@\S+\.\S+/
```

复杂版：
```s
/^([A-Za-z0-9_\-\.])+\@([A-Za-z0-9_\-\.])+\.([A-Za-z]{2,4})$/
```

### 2. 用户名
用户名正则，4到16位（字母，数字，下划线，减号）
```s
/^[a-zA-Z0-9_-]{4,16}$/
```
