### 知识简介
<https://developer.mozilla.org/en-US/docs/Web/CSS/::-webkit-scrollbar>

The ::-webkit-scrollbar CSS pseudo-element(为元素) affects the style of the scrollbar of an element

```
&.show-pushdown {
  overflow-x:scroll;

  &::-webkit-scrollbar {
      width:8px;
      height:10px;

      background:#aaa;
  }
  &::-webkit-scrollbar-thumb {
    
    background:#000;
  }
}
```
