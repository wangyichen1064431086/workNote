### 查看远程仓库信息

```cmd
git remote -v
```

### 1.去Github页面修改仓库信息

跑到自己的仓库那,找到Setting的tag, 点进去后Options的Settings就可以设定Repository name.

### 2.修改本地仓库信息

git remote set-url origin git@github.com:username/newrepo.git
修改远程仓库对应的网址.
