## 意义简介
网格布局最终被 CSS 采纳为标准，浏览器也正在快速地采用它。过去，网格系统在 CSS 中曾被 tables、float、flex 以及 inline-block 实现过。原生的 CSS 网格布局擅长于将一个页面划分成几个主要的区域，并为内容创建列和行。查看 Rachel Andrew 写的 https://gridbyexample.com/ 开始学习。
